This my Multiplayer VR project of a Dungeons and Dragons (DND) siumlator which I created in Unity using C#, Photon and the Unity VR XR toolkit, it can have up to 10 players in the server at any one time and includes in game voice chat. 
A video demo of the game can be found here - https://youtu.be/u-N5P0n7oyE
